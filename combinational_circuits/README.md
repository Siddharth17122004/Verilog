# Combinational Circuits in Verilog

This repository contains Verilog implementations of basic combinational circuits.

## 📁 Folder Structure
- src/ : Design files
- tb/  : Testbenches
- sim/ : Simulation outputs

## ⚙️ Tools Used
- Icarus Verilog (iverilog)
- GTKWave

## ▶️ Run Simulation
make run
